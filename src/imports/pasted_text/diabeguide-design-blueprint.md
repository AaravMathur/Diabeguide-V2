# DiabeGuide — Figma Design Blueprint

## 1. Project Overview

DiabeGuide is an AI-powered diabetes management platform designed to help users:

* Track blood glucose levels
* Chat with an AI health assistant
* Maintain health history
* Receive guidance and reminders
* Manage profile and lifestyle information

The platform combines:

* Diabetes tracking
* AI chatbot assistance
* User authentication
* Smart health dashboard
* Medical-style clean UI
